    Beispielanwendung 1

    Hier soll eine Zimmerpflanze überwacht werden. Hierzu kann die Feuchtigkeit, und der PH-Wert der Erde gemessen werden. Ebenso soll die Lichteinstrahlung auf die Zimmerpflanze gemesen werden.

    Benötigte Sensoren: - Soil Sensor (Bodenfeuchte) - EZO-ph (PH-Wert der Erde) - VEML7700 (Lichteinstrahlung)

    ungefährer Preis: ~60€

    Beispielanwendung 2

    Hier soll eine Wetterstation errichtet werden welche folgende Messungen durchführt:

    • Windstärke
    • Lufttemperatur
    • Luftfeuchte
    • Pollenbelastung

    Benötigte Sensoren: - Anemometer (Windstärke) - DT21 (Luftfeuchte, Lufttemperatur) - PMSA0031I (Pollenbelastung)

    ungefährer Preis: ~95€
